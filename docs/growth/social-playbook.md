# Organic Social Multiplication Playbook

## Channels
- YouTube Shorts
- TikTok

## Cadence
- 5 to 6 short videos per week

## Mandatory Creative Structure
1. 3-second hook
2. 15-30 second transformation (text -> voice output)
3. CTA to one specific landing URL

## Weekly System
1. Pull top rising queries from Search Console
2. Convert each query into one short script angle
3. Repurpose every new article into 2 to 3 social variants
4. Track retention + click-through; drop weak formats quickly

## Performance Review
- Keep only formats that retain opening attention and generate site visits
- Remove recurring formats with weak retention after 3 iterations
